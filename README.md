# PCS-SignAI
Personalized Chinese Sign Language Semantic Understanding System via Multimodal Sign Parsing, RAG and User Memory.

## Dynamic Website

项目动态介绍首页位于 `frontend/`，可独立运行

```powershell
cd frontend
npm install
npm run dev
```

类型检查、代码规范检查和生产构建：

```powershell
npm run typecheck
npm run lint
npm run build
```

更完整的前端说明见 `frontend/README.md`。
