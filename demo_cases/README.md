# Demo 视频素材

演示视频包含人物画面，因此不提交到公开仓库。请从项目负责人处取得已获授权的素材，并放到本目录；也可以在前端从其他本地目录选择文件。

Module A 的 MVP 分类器按视频文件名（不含扩展名）读取 `data/labels/demo_label_map.json`。两个核心案例必须使用以下文件名：

- `吃席.mp4`：展示中文文化知识库（RAG）命中。
- `妈妈药.mp4`：展示合成演示用户的个性化记忆命中。

扩展名也可使用 `.mov`、`.avi`、`.mkv` 或 `.webm`，但文件名主体必须保持为 `吃席` 或 `妈妈药`。视频需要能够被 OpenCV 读取，并至少有部分帧可被 MediaPipe 检出人体关键点。
